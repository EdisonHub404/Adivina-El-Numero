# Adivina-El-Numero
Proyecto Adivina el Número 
Proyecto académico desarrollado para la asignatura de lógica de programación.

Descripción:
El usuario piensa en un número entre 1 y 100 y el computador intenta adivinarlo utilizando pistas proporcionadas por el usuario.

- Diagramas de análisis y diseño.
- Avance del código fuente.
